# Tugas Pengolahan Citra - Histogram Matching

Aplikasi web sederhana untuk tugas mata kuliah Pengolahan Citra.
Project ini mengimplementasikan histogram matching pada citra grayscale dan menampilkan histogram citra asli, target, serta output.

## Fitur
- Upload citra asli
- Upload citra target
- Proses histogram matching
- Visualisasi histogram grayscale

## Teknologi
- HTML
- CSS
- JavaScript
- Chart.js
