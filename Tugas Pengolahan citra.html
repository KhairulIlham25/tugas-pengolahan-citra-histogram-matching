Informasi Singkat

Metode ini menyesuaikan distribusi histogram citra input agar mendekati histogram citra referensi.

Versi ini memakai grayscale supaya proses lebih ringan dan mudah dijelaskan saat presentasi.

1. Pilih Citra Asli

Citra ini akan diproses untuk menyesuaikan distribusi histogramnya.

2. Pilih Citra Target

Citra target menjadi acuan bentuk histogram yang ingin didekati.

3. Jalankan Proses

Tekan tombol di bawah untuk melakukan histogram matching dan menampilkan hasil visualisasi.

Output akan muncul pada bagian hasil gambar dan histogram.

Citra Asli

Gambar input sebelum dilakukan proses penyesuaian histogram.

Citra Target

Gambar referensi yang menjadi tujuan distribusi intensitas.

Citra Output

Hasil akhir setelah citra asli disesuaikan terhadap histogram target.
